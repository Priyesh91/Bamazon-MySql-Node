# Bamazon-MySql-Node

## CLI-APP with node.js

### Initial question promting user if they would like to make a purchase.
![1](https://user-images.githubusercontent.com/45401070/53819340-c60cf580-3f37-11e9-8238-dd714045c06b.PNG)

### If yes is selected, table of products from MySql database is displayed and customer is promted to enter Id of product they would like to purchase.
![1](https://user-images.githubusercontent.com/45401070/53819347-c9a07c80-3f37-11e9-9ae4-5ac3f16a1c68.PNG)

![1](https://user-images.githubusercontent.com/45401070/53819350-cad1a980-3f37-11e9-94cd-6b25f705c9ec.PNG)

### If Yes is selected another question is promted to customer asking what quantity of product they would like purchase.
![1](https://user-images.githubusercontent.com/45401070/53819353-cc02d680-3f37-11e9-9b1d-f521a34c684f.PNG)

### Once quantity is selected, customer is displayed with what they are purchasing the cost and total cost and prompted to confirm the purchase. Once purchase is confirmed the stock quantity is updated in MySql Database.
